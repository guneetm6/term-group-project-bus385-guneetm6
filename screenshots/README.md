## How to Capture Screenshots

### Microsoft Access Relationships:
1. Open the .accdb file
2. Click **Database Tools** tab → **Relationships**
3. Arrange tables for clarity (drag headers)
4. Use Windows Snipping Tool (Win+Shift+S) or Print Screen
5. Save as `access_relationships.png` in this folder

### MySQL Workbench Query Results:
1. Open the .sql file in MySQL Workbench
2. Place cursor in the query, press Ctrl+Enter to execute
3. With both the SQL pane (top) and result grid (bottom) visible, take a screenshot
4. Save with descriptive filename
